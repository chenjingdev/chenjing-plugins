**변경**:
- SKILL.md "AskUserQuestion 변환 규칙" (line 131~) 7번을 확장:
  ```
  7. **서술형 질문 감지 (확장)**:
     다음 유형은 `N)` 패턴이 있어도 AskUserQuestion을 호출하지 않고 **평문 텍스트로 전달**한다:
  ```
