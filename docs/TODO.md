## 인터뷰 프로세스 강화

1. **So What 체인** — 에피소드 저장 시 임팩트를 한 단계 더 깊이 파는 후속 질문 트리거. "파이프라인 구축" → "배포 시간 절감" → "팀 생산성 향상" 식으로 비즈니스 임팩트까지 도달.

2. **패턴 발견** — 에피소드가 쌓이면 프로파일러가 크로스 컴퍼니 패턴을 탐지. "가는 곳마다 자동화를 밀었네? C회사에서도 비슷한 거 했을 텐데" → 숨은 에피소드 발굴.

3. **타임라인 갭 탐지** — 회사/프로젝트 기간 정보로 타임라인을 구성하고, 빈 구간을 자동으로 찾아서 프로빙. "A 프로젝트 끝나고 B 시작까지 6개월인데 그때 뭐 했어?"

4. **관점 전환 질문** — 같은 에피소드를 타인 시점에서 묻기. "네 PM이 이 프로젝트에서 네 역할을 설명한다면?", "주니어 팀원 입장에서 보면?" → 본인 시점에서 안 보이던 임팩트 발견. 리더십/협업 에피소드에서 특히 강력.

5. **모순 탐지** — 인터뷰 중 앞뒤가 안 맞는 답변 캐치. "아까 관여 안 했다고 했는데 방금 재설계했다고 했잖아" → 겸손해서 축소한 에피소드를 잡아냄. 한국 사람들이 본인 역할을 축소하는 경향이 있어서 효과적.
