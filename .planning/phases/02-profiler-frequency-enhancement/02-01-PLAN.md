<objective>
Replace episode-watcher.mjs's simple delta-based profiler trigger (episodeDelta >= 3 + cooldown) with
an event-weighted scoring system. Each event type adds a specific weight to a cumulative score stored
in meta.json. When the score reaches threshold 5, the hook outputs a profiler trigger message and
resets the score to 0.

Purpose: Strategic profiler timing -- a single new company (+3) or result gap (+2) + meta change (+2)
triggers the profiler immediately, while routine episode saves (+1 each) trigger it after 5 episodes.
This replaces the blunt "every 3 episodes" rule with context-aware triggering.

Output: Modified episode-watcher.mjs with scoring system, updated test file with full coverage.
</objective>

1. Add a new helper function `detectMinimization(source, snapshot)` before the messages array:
   ```javascript
   function detectMinimization(source, snapshot) {
     const MINIMIZATION_KEYWORDS = ["도움", "참여", "지원", "보조", "서포트"];
     const prevCount = snapshot.episode_count || 0;
     let found = false;
     let checked = 0;
     for (const project of source.projects || []) {
       for (const ep of project.episodes || []) {
         checked++;
         if (checked <= prevCount) continue; // skip already-seen episodes
         const text = `${ep.star?.action || ""} ${ep.star?.result || ""}`;
         if (MINIMIZATION_KEYWORDS.some(kw => text.includes(kw))) {
           found = true;
           break;
         }
       }
       if (found) break;
     }
     return found;
   }
   ```

   ```javascript
   // Score calculation
   const metaJSON = readJSON(metaPath) || {};
   let score = metaJSON.profiler_score || 0;
   const reasons = [];

   // +1: episode save
   const episodeDelta = currentCount - (snapshot.episode_count || 0);
   if (episodeDelta > 0) {
     score += episodeDelta; // +1 per new episode
     reasons.push(`에피소드 +${episodeDelta}`);
   }

   // +3: new company/project
   const snapshotProjects = new Set(snapshot.project_names || []);
   const hasNewProject = currentProjects.some(p => !snapshotProjects.has(p));
   if (hasNewProject) {
     score += 3;
     reasons.push("새 프로젝트 (+3)");
   }

   // +2: new star gaps (empty result)
   const currentStarGaps = countStarGaps(source);
   const prevStarGaps = snapshot.star_gaps || 0;
   if (currentStarGaps > prevStarGaps) {
     score += 2;
     reasons.push(`빈 STAR 증가 (+2)`);
   }

   // +2: role minimization signal
   if (detectMinimization(source, snapshot)) {
     score += 2;
     reasons.push("역할 축소 신호 (+2)");
   }

   // +2: meta change
   if (currentHash !== snapshot.meta_hash) {
     score += 2;
     reasons.push("meta 변경 (+2)");
   }
   ```
